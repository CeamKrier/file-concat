# GitHub Import

Import any public GitHub repository by pasting its URL.

## Supported Formats

```
https://github.com/owner/repo
https://github.com/owner/repo/tree/main
https://github.com/owner/repo/tree/main/packages/core
```

## Tips

- Large repositories may take a bit longer to fetch.
- Use filters after import to narrow the output.
- Check the token count before copying.
